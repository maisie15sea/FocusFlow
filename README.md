## Overview

FocusFlow is a sleek and minimal Pomodoro timer designed to help you optimize your work sessions and maintain concentration. It features task management, customizable work and break intervals, and historical progress tracking to keep you motivated.

## Features

- Customizable Pomodoro intervals (work, short break, long break)
- Simple task creation and tracking
- Session history with productivity statistics
- Intuitive and distraction-free interface
- Desktop notifications for session transitions

## Tech Stack

- React.js for the frontend
- Redux for state management
- Tailwind CSS for styling
- LocalStorage for data persistence

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/maisie15sea/FocusFlow.git
   ```

2. Navigate into the project directory:

   ```bash
   cd FocusFlow
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Start the development server:

   ```bash
   npm start
   ```

## Usage

- Open the app in your browser at `http://localhost:3000`.
- Create a task to work on.
- Set your preferred Pomodoro intervals in settings.
- Start the timer and focus during work sessions.
- Take breaks when the timer notifies you.
- Review your session history to track productivity over time.
